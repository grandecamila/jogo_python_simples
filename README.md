# jogo_python_simples\
Jogo simples em python\
Coloque o código em um interpretador para poder jogar, tipo o REPLIT\
Utilize opção 1 para iniciar o jogo\
Utilize as setas direita e esquerda para se movimentar no jogo\
Utilize a tecla de espaço ou W para pular\
Utilize a tecla R para reiniciar o jogo
